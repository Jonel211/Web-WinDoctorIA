WinDoctorIA

Sistema de Diagnóstico y Reparación Inteligente para Windows

Versión 1.0.0

Resumen Ejecutivo

WinDoctorIA es un sistema inteligente de diagnóstico y reparación automática para sistemas operativos Windows, desarrollado con python. El sistema permite detectar, diagnosticar y resolver problemas de rendimiento (CPU, RAM, disco, red) de forma automática y segura, reduciendo significativamente los tiempos de resolución.

2. Objetivos del Proyecto

2.1 Objetivo General

Desarrollar un sistema inteligente de diagnóstico y reparación automática para sistemas operativos Windows para detectar, analizar y resolver problemas de rendimiento de forma segura, eficiente y sin intervención manual del usuario.

2.2 Objetivos Específicos

  1. Implementar un motor de diagnóstico inteligente capaz de analizar en tiempo real el estado del sistema (CPU, RAM, disco, red, temperatura) y detectar anomalías de rendimiento.
  2. Desarrollar un Gestor de Inicio que optimice el arranque de Windows y carpetas de inicio.
  3. Implementar un Buscador de Duplicados eficiente utilizando algoritmo SHA-256 en 3 etapas (tamaño, hash parcial, hash completo) con integración nativa a la Papelera de Reciclaje mediante ctypes.
  4. Generar reportes profesionales en formatos PDF y TXT que documenten el estado del sistema, problemas detectados y acciones realizadas.

2.3 Alcance del Proyecto

Incluido

  • Diagnóstico de CPU, RAM, disco, red y temperatura
  • Buscador de Duplicados con hashing SHA-256
  • Dashboard con gráficos GPU accelerated
  • Integración nativa con API de Windows (ctypes)
  • Generación de reportes PDF/TXT
  • Interfaz gráfica moderna con PySide6
  • Soporte para Windows 10/11 (64-bit)

No Incluido

  • Soporte para Windows 7/8
  • Versión para macOS o Linux
  • Antivirus completo en tiempo real
  • Reparación de hardware físico
  • Soporte remoto o cloud

3. Problemática

3.1 Contexto del Problema

Los sistemas operativos Windows, siendo los más utilizados a nivel mundial con más del 70% de cuota de mercado en computadoras de escritorio, enfrentan problemas recurrentes de rendimiento que afectan la productividad de usuarios individuales y empresas.

Según estudios de la industria, los usuarios de Windows experimentan en promedio:

  • 3-5 problemas de rendimiento por mes
  • 2-4 horas perdidas mensualmente en diagnóstico y resolución
  • Falta de monitoreo proactivo del estado del sistema
  • Difícil acceso a reportes técnicos profesionales

3.2 Problemas Identificados

Alto Uso de CPU

Procesos en segundo plano consumen recursos excesivos, causando lentitud general del sistema.

Memoria RAM Insuficiente

Aplicaciones acumulan memoria sin liberarla, provocando ralentización y cierres inesperados.

Disco Lleno

Archivos temporales y caché acumulados ocupan espacio crítico del disco duro.

Problemas de Red

Configuración de red corrupta impide conexión a internet o causa intermitencias.

3.3 Barreras Actuales

Desafíos para los Usuarios
  • Conocimiento técnico requerido: La mayoría de soluciones requieren experiencia en sistemas
  • Herramientas dispersas: No existe una solución unificada para todos los problemas
  • Riesgo de errores: Acciones incorrectas pueden empeorar el sistema
  • Tiempo de diagnóstico: Identificar la causa raíz puede tomar horas

5.2 Componentes Principales

Capa de Presentación

  • LayoutMain: Ventana principal con navegación lateral
  • PanelInicio: Dashboard con gráficos circulares profesionales (gradientes cónicos)
  • PanelDiagnostico: Visualización de problemas detectados
  • PanelAsistente: Interfaz de IA conversacional
  • PanelReparaciones: Gestor de Inicio con diálogo de programas
  • PanelMonitor: Monitoreo en tiempo real con widgets personalizados
  • PanelDuplicados: Buscador de archivos duplicados
  • PanelReportes: Generación de documentos PDF/TXT

Capa de Lógica

  • AsistenteIA: Motor de inteligencia artificial
  • Sistema: Recopilación de métricas (psutil)
  • Reparador: Gestión de inicio, papelera, caché y elevación UAC
  • Duplicados: Detección de duplicados con SHA-256 en 3 etapas
  • Monitor: Servicio de monitoreo en tiempo real
  • GeneradorPDF: Creación de reportes profesionales

Capa de Datos e Integración

  • psutil: Recopilación de métricas del sistema (CPU, RAM, Disco, Red, Temp)
  • ctypes + Windows API: SHFileOperationW (Papelera), ShellExecuteW (UAC), SHQueryRecycleBinW
  • winreg: Lectura/escritura del Registro de Windows
  • pythoncom + win32com: Resolución de accesos directos .lnk
  • hashlib: Hashing SHA-256 para detección de duplicados
  • os/shutil: Manipulación de archivos y directorios

7. Funcionalidades Principales

7.1 Módulos del Sistema

01

Diagnóstico Inteligente

Análisis en tiempo real de CPU, RAM, disco, red y temperatura con detección automática de anomalías.

02

Gestor de Inicio

Optimización del arranque de Windows mediante escaneo exhaustivo del Registro y carpetas de inicio.

03

Buscador de Duplicados

Detección eficiente de archivos duplicados mediante algoritmo SHA-256 en 3 etapas (tamaño, hash parcial, hash completo) con integración nativa a la Papelera de Reciclaje vía ctypes.

04

Monitoreo Visual

Dashboard con gráficos de alto rendimiento (GPU accelerated) usando QPainter, gradientes cónicos y efectos modernos para visualización de métricas en tiempo real.

05

Asistente IA

Interfaz conversacional que analiza el estado del sistema y sugiere soluciones personalizadas basadas en el contexto actual.

06

Reportes Profesionales

Generación automática de documentos PDF/TXT con análisis completo del sistema y registro detallado de todas las acciones realizadas.
